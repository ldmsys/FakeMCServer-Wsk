# FakeMCServer-WSK
Stub MC Server that runs on your NT Kernel. Providing basic kick and MOTD behavior.

## Video in action
[![.](https://i.ytimg.com/vi/dkCHKAsWinE/hqdefault.jpg)](https://youtu.be/dkCHKAsWinE)

## Features
 * Uses lesser RAM
 * Can set custom MOTD
 * Kick immediately after user join
